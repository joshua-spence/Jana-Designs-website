Spekk

This project started with extensive experiments for my exploration to familiarise myself with the characteristics and properties of my chosen materials - concreate and plaster, my finding helped me develop a household tool or object. I explored ways to incorporated waste building materials such as bricks, slate, aerated concrete, and tiles, to cement and plaster mixes to create an appealing speckled aesthetic within. I also investigated ways to generate organic forms with the moulds for the concrete. Using fabrics resulted in organic shapes and detailed texture transferring to the concrete.

grinder

Spekk is a pestle & mortar set used to grind herbs and spices. The set would be cast in cotton so the fine detail of the woven fabric would transfer to the concrete and therefore create friction when grinding. The mortar was designed to be cast in a sack- like cast which resembles a wheat sack to maintain coherence.
